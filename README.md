# ゲーム のタイトル
## 実行環境の必要条件
* python >= 3.10
* pygame >= 2.1

## ゲームの概要
主人公キャラクターのダニエルをキーボードを使って操作するゲームで、アイテム等を使い敵を倒したりしステージをクリアするゲーム。

## ゲームの実装
###共通基本機能
* 背景画面スクロールに関するクラス
* 主人公に関するクラス
### 担当追加機能
* 敵キャラクター
* オブジェクト
* アイテム①
* アイテム②
* コイン
